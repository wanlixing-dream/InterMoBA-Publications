# Journal / venue notes — index

This directory holds one Markdown file per submission target, capturing the venue''s hard requirements, formatting rules, deadlines, and a submission checklist.

| File | Target | Status |
| --- | --- | --- |
| `apbc2026.md` | APBC2026 (MDPI) | preparing — primary |

To add a new target:

1. Create `docs/journal-notes/<venue>.md` with the venue name, deadline, formatting rules, and a checklist.
2. Add a row to the table above.
3. Add a `manuscripts/<venue>-<short>/` skeleton next to `manuscripts/mdpi-apbc2026/`.
4. Append a row to the top-level `README.md` "Submission status" table.
