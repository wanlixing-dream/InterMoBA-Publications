## 关键要求 (从模板抽取)

### 主稿
- 单段 structured abstract, **≤ 250 词**, 含 Background / Methods / Results / Conclusions
- 3 – 10 个 keywords
- 主文 (排除 abstract / 图注 / Author Contributions / 致谢 / 参考文献) **≤ 5000 词**
- 参考文献: 顺序编号制 `[1]`, `[1-3]`, `[5] (p. 10)`, 数字在方括号内、句点前
- 行号 (`lineno`) 全程开启 — 模板默认开
- A4, 11pt, 2.5 cm 边距
- 作者单位用 `authblk` 体系 — 见 `author-info.tex`

### Supplementary
- **独立 PDF** — 不要嵌进主稿
- MDPI 投稿时打成 `.zip` 一并上传
- 文件名: `InterMoBA_supp.pdf` (或会议约定)

### 图表
- 图优先用 `.pdf` / `.eps` (矢量) — 现有 `.png` 满足最低要求
- 引用 `figures/fig{n}.png` (主稿) / `figures/{原名}.png` (supplementary)

## 提交 checklist

- [ ] 选定目标 MDPI 子刊
- [ ] 确认审稿政策 (单盲 / 双盲)
- [ ] 填齐 `author-info.tex` (作者 / 单位 / 通讯)
- [ ] 写主稿 `main.tex` 正文 (Abstract / Intro / Methods / Results / Discussion / Conclusions)
- [ ] 生成 `main.pdf` (本地编译)
- [ ] 生成 `supp.pdf` (本地编译)
- [ ] 把 `supplementary/` 打成 `.zip`
- [ ] 登录 MDPI 投稿系统, 上传 main.pdf + supp.zip + 任何 cover letter
- [ ] 在 `docs/submission-log.md` 追加一行记录 (日期 / 阶段 / 状态)
