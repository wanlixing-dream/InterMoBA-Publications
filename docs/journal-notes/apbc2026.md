## Current Working-Manuscript Requirements

These are extracted from the local APBC/MDPI-style template and should be treated as working constraints until APBC assigns the final journal/template.

### Main Manuscript

- Single structured abstract, target <= 250 words, with Background / Methods / Results / Conclusions.
- 3-10 keywords.
- Main text target <= 5000 words, excluding abstract, captions, author contributions, acknowledgements, and references.
- Numbered reference style in the local skeleton; final bibliography style may change after journal assignment.
- Line numbers enabled with `lineno`.
- A4, 11 pt, 2.5 cm margins.
- Author block uses `authblk`; author details live in `author-info.tex`.

### Supplementary

- Keep supplementary material as an independent PDF; do not embed it into the main manuscript.
- Package supplementary material as a separate archive for submission.
- Proposed final file name: `InterMoBA_supp.pdf`, unless APBC or the assigned journal specifies otherwise.

### Figures and Tables

- Prefer PDF/EPS vector graphics after final journal assignment if available; current PNG figures are retained as minimum viable assets.
- Main manuscript figures use `figures/fig{n}.png`.
- Supplementary figures use `supplementary/figures/{name}.png`.
